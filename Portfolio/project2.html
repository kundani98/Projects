About My project

I am excited to present to you my Movie Review App Project.This project is a basic MERN stack where clients can create reviews about diffent movies they have watched.It utilizes the following technologies:MongoDB,Express,React and Node.Js. This Project demostrate my ability to use MongoDB as my database, creating a Node.js Server and connecting to the backend using a React front end.Movie reviews app lets users view and search for movies. They can also log in and post reviews on the movies.
